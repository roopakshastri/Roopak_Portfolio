FAVICON PLACEHOLDER INSTRUCTIONS
=================================

A favicon.ico file should exist in the root of your portfolio.

WHAT IS A FAVICON?
------------------
The small icon that appears in browser tabs, bookmarks, and history.
Example: The "G" icon for Google.com

CURRENT STATE:
--------------
The index.html references /favicon.ico, but this file doesn't exist yet.
The site will work fine without it, but you'll see a broken icon or
default browser icon.

OPTION 2: Use RealFaviconGenerator.net
---------------------------------------
1. Go to https://realfavicongenerator.net/
2. Upload your logo or design (512x512px PNG recommended)
3. Customize for different platforms (iOS, Android, Windows)
4. Download the package
5. Copy all files to your portfolio root
6. Update <head> section with provided code

OPTION 3: Create Manually
--------------------------
Using GIMP or Photoshop:
1. Create a 32x32px image (or 64x64px for retina)
2. Design your icon
3. Export as .ico format
4. Name it favicon.ico
5. Place in portfolio root

Using Online Converters:
1. Create a PNG image (256x256px or 512x512px)
2. Go to https://convertio.co/png-ico/
3. Upload your PNG
4. Download the .ico file
5. Rename to favicon.ico if needed

COMPLETE FAVICON PACKAGE (Advanced):
-------------------------------------
For best compatibility across all devices, include these files:

/favicon.ico             (32x32 for browsers)
/favicon-16x16.png       (16x16 for browser tabs)
/favicon-32x32.png       (32x32 for browser tabs)
/apple-touch-icon.png    (180x180 for iOS)
/android-chrome-192x192.png (192x192 for Android)
/android-chrome-512x512.png (512x512 for Android)
/site.webmanifest        (manifest for PWA)

Then update <head> in index.html:

<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">

TESTING YOUR FAVICON:
---------------------
1. Place favicon.ico in root folder
2. Commit and push to GitHub
3. Wait 1-2 minutes for deployment
4. Visit your site in a new incognito/private window
5. Check browser tab for your icon
6. Hard refresh if needed (Ctrl+Shift+R or Cmd+Shift+R)

TROUBLESHOOTING:
----------------
Icon not showing?
- Clear browser cache
- Try incognito/private mode
- Ensure file is named exactly: favicon.ico
- Verify file is in root directory (not in /images/)
- Check file size is reasonable (<100KB)
- Wait 5-10 minutes for browser cache to clear

Still not working?
- Check browser console (F12) for 404 errors
- Verify path in <link> tag is correct
- Ensure MIME type is correct (image/x-icon)

NO FAVICON IS OKAY:
-------------------
Your portfolio will function perfectly without a favicon.
It's a nice-to-have, not a requirement.
You can add it later without affecting functionality.
