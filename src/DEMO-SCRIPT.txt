COMMON QUESTIONS & ANSWERS
===============================================================================

Q: "Why no frameworks like React or Vue?"
A: "To minimize dependencies and attack surface. Vanilla JavaScript is more
    secure as there's no risk of framework vulnerabilities, and the site
    loads faster without heavy framework overhead."

Q: "How do you handle form submissions without a backend?"
A: "Currently uses mailto for demonstration. In production, I'd implement
    a serverless function or integrate with Netlify Forms for server-side
    validation and processing."

Q: "What's your deployment process?"
A: "Push to GitHub triggers automatic deployment. Changes go live in under
    2 minutes. The workflow is: code â†’ git commit â†’ git push â†’ automatic deploy."

Q: "How did you achieve A+ security rating?"
A: "By implementing all recommended security headers: CSP, X-Frame-Options,
    HSTS, X-Content-Type-Options, and Referrer-Policy. Each header provides
    a specific security control."

Q: "Is this production-ready?"

===============================================================================
VIDEO RECORDING TIPS
===============================================================================

TECHNICAL SETUP:
----------------
- Screen resolution: 1920x1080 (Full HD)
- Recording software: OBS Studio, Loom, or QuickTime
- Frame rate: 30fps minimum
- Audio: Clear microphone, minimize background noise
- Browser zoom: 100% (don't zoom in/out)

RECORDING BEST PRACTICES:
--------------------------
1. Close unnecessary browser tabs
2. Clear browser history/cache
3. Disable browser extensions (may interfere)
4. Use incognito/private mode for clean interface
5. Hide bookmark bar for cleaner screen
6. Prepare all URLs in advance
7. Do a practice run before recording

POST-PRODUCTION:
----------------
- Trim dead time at start/end
- Add title card with your name
- Include URL overlay of deployed site
- Add annotations for key features
- Export at 1080p resolution
- Keep file size under 100MB for easy submission
