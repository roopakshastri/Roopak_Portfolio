4. CHALLENGES & SOLUTIONS
========================================

Challenge 1: Implementing project filtering without JavaScript frameworks
Solution: Used data attributes (data-tags) and classList manipulation to show/hide projects based on selected filter

Challenge 2: Ensuring accessibility for modal dialog
Solution: Implemented focus management, keyboard controls (Escape to close), and ARIA attributes (aria-modal, role="dialog")

Challenge 3: Contact form without backend server
Solution: Used mailto: link generation with URL-encoded subject/body as fallback, with client-side validation

Challenge 4: CSP blocking inline styles needed for Google Fonts
Solution: Added 'unsafe-inline' to style-src directive (acceptable for trusted first-party styles) and whitelisted fonts.googleapis.com
