========================================
SECURITY FEATURES
========================================

✓ Content Security Policy (CSP) meta tag
  - Restricts scripts to same origin
  - Whitelists Google Fonts only
  - Blocks inline scripts (unsafe-inline not used for JS)
  - Prevents clickjacking (frame-ancestors 'none')
  
✓ HTTPS enforcement (automatic on GitHub Pages & Netlify)

✓ No external JavaScript libraries (zero third-party attack surface)

✓ Form validation prevents XSS via input sanitization

✓ No credentials or API keys in client code

✓ Security headers snippet for Netlify (_headers file in deploy-instructions.txt)

✓ Educational console warning about browser dev tools scams
