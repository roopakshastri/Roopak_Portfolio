FILE PERMISSIONS (FOR UNIX SERVERS)
===============================================================================

If deploying to Linux/Unix server, set permissions:

chmod 644 index.html
chmod 644 css/style.css
chmod 644 js/main.js
chmod 644 manifest.json
chmod 644 _headers
chmod 644 LICENSE
chmod 755 images/
chmod 755 docs/
chmod 755 css/
chmod 755 js/

Or simply:
find . -type f -exec chmod 644 {} \;
find . -type d -exec chmod 755 {} \;

GitHub Pages/Netlify: Permissions handled automatically
